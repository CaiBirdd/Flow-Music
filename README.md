# Flow Music

FlowMusic 是一款基于 Electron 和 Vue 3 构建的沉浸式桌面音乐播放器。支持 LRC 双语歌词匹配与同步滚动、根据单曲封面自适应提取并渲染双缓冲流态背景，以及全局音频的平滑淡入淡出过渡，为用户提供了流畅的音画体验。

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
